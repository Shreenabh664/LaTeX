	\textbf{Solution:} After squaring,$$ \implies 2\sum \sqrt {\frac {ab}{(b + c)(a+ c)}}\le3$$ By AM-GM, $$ LHS\le\sum\left( \frac a{a + c} + \frac b{b + c}\right) = 3$$

	\textbf{Solution:} Let $ x=a,y=b^2,z=c^3$, ineq becomes $$ x+y+z=1,\,\,(1-x)(1-y)(1-z)\ge 8xyz$$ i.e. $$ (x+y)(y+z)(z+x)\ge8xyz$$ which is true

	\item Let $ a,b,c$ be real positive numbers and $ a+b+c=1$, Prove that$$2\geq\frac{(1-a)^3}{3a+b+c}+\frac{(1-b)^3}{3b+c+a}+\frac{(1-c)^3}{3c+a+b}\geq\frac{8}{15}$$ 
	
	\textbf{Solution:}  By Cauchy, we have
	\[ \sum \frac{(b+c)^4}{(b+c)(b+c+3a)} \ge \frac{\left( \sum (b+c)^2 \right)^2}{\sum (b+c)(b+c+3a)} \ge \frac{15}{8}.\]
	Denote $ \sum a^2=x$, $ \sum ab=y$. Then the inequality is equivalent to\[ 9x^2+6xy \ge 17y^2,\]which is obvious, as $ \sum a^2 \ge \sum ab$ with equality if and only if $ a=b=c$.
	
	We have $ 3a+b+c \ge a+b+c$ and so on. Hence
	\[ \sum \frac{(b+c)^3}{b+c+3a} \le \sum \frac{(b+c)^3}{a+b+c} \le 2\Leftrightarrow 3\sum (a^2b+ab^2) + 12abc \ge 0,\]which is true. Equality holds if and only if $ a=b=0$, $ c=1$ and permutations.

	\textbf{Solution:} For the left :Use Cauchy-Schwarz ineq, we have:\begin{align*}\sqrt {a + {b^2} + {c^2}}  + \sqrt {b + {c^2} + {a^2}}  + \sqrt {c + {a^2} + {b^2}}& \le \sqrt {3(a + b + c + 2({a^2} + {b^2} + {c^2}))}\\& \le \sqrt {3(a + b + c + {2{(a + b + c)}^2})}= 3\end{align*}Equality holds when (a,b,c)=(1,0,0)\\For the right : Use Mincowski ineq, we have:\begin{align*}&{(\sqrt {a + {b^2} + {c^2}}  + \sqrt {b + {c^2} + {a^2}}  + \sqrt {c + {a^2} + {b^2}} )^2}\\ =&\,  a + b + c + 2({a^2} + {b^2} + {c^2}) + 2\sum {\sqrt {(a + {b^2} + {c^2})(b + {c^2} + {a^2})} }\\ = &\, 1 + 2({a^2} + {b^2} + {c^2}) + 2\sum {\sqrt {({a^2} + {b^2} + {c^2} + a(b + c))({b^2} + {c^2} + {a^2} + b(c + a))} }\\ \ge &\, 1 + 2({a^2} + {b^2} + {c^2}) + 2\sum {({a^2} + {b^2} + {c^2} + \sqrt {ab(c + a)(c + b)} } ) \\  = &\, 1 + 8({a^2} + {b^2} + {c^2}) + 2\sum {\sqrt {ab(c + a)(c + b)} } \end{align*}We need to prove:\begin{align*}& 1 + 8({a^2} + {b^2} + {c^2}) + 2\sum {\sqrt {ab(c + a)(c + b)} }  \ge  5\\ \iff &\, 2({a^2} + {b^2} + {c^2}) + \sum {\sqrt {ab(c + a)(c + b)} } \ge 4(ab + bc + ca) \end{align*}By AM-GM ineq , we have:$$\sum {\sqrt {ab(c + a)(c + b)} }  \ge 2\sum {\sqrt {abc\sqrt {ab} } }  \ge 6\sqrt[3]{{{a^2}{b^2}{c^2}}} \ge \frac{{18abc}}{{a + b + c}}$$So this ineq can be rewriten:\[{a^2} + {b^2} + {c^2} + \frac{{9abc}}{{a + b + c}} \ge 2(ab + bc + ca)\]It's Schur inequality\\Equality holds when $a=b=c=\frac{1}{3}$
